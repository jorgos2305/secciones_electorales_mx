# Mexican voting sections

This program helps create maps of the voting sections in Mexico.
The data is openly available on the INE's web page: https://cartografia.ine.mx/sige8/

Two inputs are necessary:

1. *.shp files contain the geometry data needed to create the map
2. The names of all the regions (municipios) in the federal state
